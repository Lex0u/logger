# Politique de sécurité

## Versions supportées

| Version | Supportée |
| ------- | --------- |
| Dernière version majeure | ✅ |
| Versions antérieures | ❌ |

## Signaler une vulnérabilité

Si tu découvres une faille de sécurité dans `@lex0u/logger`, merci de **ne pas** ouvrir une issue publique.

Utilise plutôt l'onglet **Security → Report a vulnerability** de ce dépôt (private vulnerability reporting), ou contacte directement le mainteneur via le profil GitHub [@Lex0u](https://github.com/Lex0u).

Merci d'inclure :

- Une description du problème
- Les étapes de reproduction
- L'impact potentiel

Un correctif ou un retour sera apporté dans les meilleurs délais.
